## Maintenance Guidelines

### Adding New Images
1. **Naming:** Use lowercase with hyphens (e.g., `new-image-name.jpg`)
2. **Optimization:** Keep file size under 300 KB for featured images, under 150 KB for supporting images
3. **Format:** Use JPG for photos, PNG for graphics/screenshots
4. **Alt text:** Add descriptive alt text in all three languages
5. **Update this README:** Add new image to inventory section

### Removing Images
1. **Check usage:** Search all HTML and JSON files for references
2. **Update references:** Remove or update image references in articles
3. **Delete file:** Remove from `/images/blog/` directory
4. **Update this README:** Remove from inventory

### Replacing Images
1. **Keep filename:** Use same filename for seamless replacement
2. **Match dimensions:** Keep similar aspect ratio to original
3. **Test all languages:** Verify image appears correctly in EN/NL/FR versions
4. **Update alt text:** If content changes, update alt text accordingly

### Checking for Duplicates
```bash
# From images/blog/ directory:
md5 -r *.jpg *.png | sort

# Look for duplicate hash values
```
