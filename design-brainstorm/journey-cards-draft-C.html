Design notes — Draft C

  • No whitespace mismatch: all cards are identical height via aspect-ratio: 3/4; text length no longer affects card height.
  • Grid: 3 cards top row, 2 cards centred on bottom row — keeps Absolute Beginners prominent as the anchor card.
  • Overlay gradient: linear-gradient(to top, rgba(28,36,75,0.95) → rgba(28,36,75,0.60) → transparent) — dark navy matches brand.
  • Hover: image scales to 1.05×, overlay deepens slightly, CTA arrow nudges right.
  • CTA style: gold text link with arrow — replaces heavy yellow button, feels lighter and more editorial.
  • Badge: gold pill retained, positioned top-left, does not compete with text overlay.
  • Responsive: 2-col on tablet (≤900px), 1-col on mobile (≤540px); mobile cards switch to 4:3 aspect ratio for better thumb scrollability.
  • Image placeholders: warm tango gradient with filename watermark — swap for real <img> tags with object-fit: cover in production.
