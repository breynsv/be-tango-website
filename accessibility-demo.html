BE-TANGO Accessibility Features Demo

1. Skip-to-Content Link

How to test: Press Tab when page loads. The skip link will appear at the top of the page. Press Enter to jump to main content.

The skip-to-content link is hidden by default but appears when focused. This allows keyboard users to bypass repetitive navigation.

2. Focus-Visible Indicators

How to test: Press Tab to navigate through these elements. You'll see a gold outline (3px) around the focused element.
Link Button

3. ARIA Labels on Icon Buttons

How to test: Use a screen reader (NVDA, VoiceOver) to navigate these buttons. They will announce their purpose.

4. Form with Labels

How to test: Click on a label - it will focus the associated input field. Screen readers will announce the label when the field is focused.

5. Keyboard Navigation Tips

  • Tab: Move to next interactive element
  • Shift + Tab: Move to previous interactive element
  • Enter/Space: Activate buttons and links
  • Arrow keys: Navigate within dropdown menus
  • Esc: Close dropdown menus

6. Screen Reader Testing

Windows: Download NVDA (free) from nvaccess.org

Mac: Use built-in VoiceOver (Cmd + F5)

Mobile: iOS VoiceOver or Android TalkBack
